\section{Background}

    \textit{Deep learning} is a subset of machine learning algorithms based on neural networks with many layers. Training is done by passing a data sample 
    through the entire network layer by layer using the output of a previous layer as the input for the next one. The output of the final layer is a prediction of the result
    and the differences, given by a chosen loss function, between this value and the ground, real truth are called gradients. This process is called the forward pass. 
    The computed gradients are then used to update the weights of all the layers, using an optimization algorithm. This step is called backpropagation.
    In each epoch the entire dataset is iterated through only once with the goal of converging to a minimum which represents all of the data samples (gradient descent), \\
    To accelerate the training process, computation can be distributed among several workers, this is called Distributed Parallel learning and involves cloning
    the model parameters among all the workers, each of the individual iterates through a subset of the data, the gradients are then averaged and shared with all the workers.
    This way, all replicas of the network are updated with the same gradients ensuring all workers remain consistet with each other at the beginning of a new iteration.
